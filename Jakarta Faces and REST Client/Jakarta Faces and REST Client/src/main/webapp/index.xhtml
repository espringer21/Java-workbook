Assignment description

This assignment helps develop my skills by creating a web application using Jakarta Faces and the Microprofile REST Client. The tasks involve implementing File Templates in IntelliJ IDEA, building a REST Client for a Firebase Realtime Database, and constructing a Jakarta Faces web application that communicates with the database via a REST API Client. Additionally, I have to convert a Blazor WebApp Razor Component into a Jakarta Faces page. The goal is to manage a dataset related to a chosen subject with at least three data fields, including one of type String and one of a non-String data type.
